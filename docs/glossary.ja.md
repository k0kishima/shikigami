# ユビキタス言語一覧

<p align='center'>
  <a href='./glossary.md'>English</a> | 日本語
</p>

| 用語 | 説明 |
|------|------|
| ロール | AIエージェントに割り当てる役割。 |
| オーケストレーター | ユーザーとのインターフェースを担うロール。要件分析を行い、タスクフォースを生成する。 |
| タスクフォース | 特定の要件を達成するために動的に生成されるAIエージェントのチーム。オーケストレーターによって設計され、要件に最適なレイヤー構成とロール配置を持つ。 |
| レイヤー | タスクフォース内の階層構造における1つの層。各レイヤーは探索・評価・意思決定などの役割を持ち、役割に応じたtemperatureが設定される。 |
