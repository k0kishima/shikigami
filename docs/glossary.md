# Glossary

<p align='center'>
  English | <a href='./glossary.ja.md'>日本語</a>
</p>

| Term | Description |
|------|-------------|
| Role | A responsibility assigned to an AI agent. |
| Orchestrator | A role that serves as the interface with users. Performs requirements analysis and generates task forces. |
| Task Force | A team of AI agents dynamically generated to achieve specific requirements. Designed by the orchestrator with optimal layer structure and role placement for the requirements. |
| Layer | A single tier within the hierarchical structure of a task force. Each layer has a responsibility such as exploration, evaluation, or decision-making, with temperature set according to its role. |
