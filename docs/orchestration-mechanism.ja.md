## Claude Code Agent Teams の採用

Shikigami のオーケストレーションは [Claude Code](https://claude.ai/code) の **Agent Teams** 機能を基盤としています。

Agent Teams は、自然言語の指示だけで AI エージェントチームを生成できるという画期的な機能を備えています。  
この機能を活用することで、オーケストレーションエンジンをスクラッチで実装することなく、動的なタスクフォース生成を実現しています。

## トレードオフ

このアプローチは Claude Code への強い依存を生みます。  
しかし、実装の複雑さを大幅に削減できるというメリットとのトレードオフとして受け入れています。
